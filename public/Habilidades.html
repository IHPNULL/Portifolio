ferramentas

Fusion 360

Fusion 360 é uma feramenta da empresa autodesk muito utilizado para projetos de peças tanto 3D quanto 2D. O software é mito utlizado para fazer simulaçoes de resistencia, aerodinamica,densidade e etc.

3DsMax

Da empresa autodesk o 3Dsmax é uma ferramenta de escupir obejtos em 3D, utiizei para fazer os modelos do primeiro e do segundo semestre da faculdade.

UNITY

Unity é uma plataforma utiliada para criaçao e desenvolvimento de Games e aplicativos, utilizei ela em ambos projetos de criaço de cenario e de jogos. compativel com C# é muito util e muito boa para praticar programaçao na linguagem.

Proteus

Proteus é uma ferramenta para desenvolver placas de circuito, muito boa para quando precisamos imprimir as placas ou usinar elas.

GIT

GIT é uma ferramenta otima para verssionamento de codigos e projetos.

Office

Office é a plataforma da microsoft para criaçao de documentos de texto, planilhas , apresentaçoes e etc.
